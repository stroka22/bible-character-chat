Repository Layer

Add these utility functions to src/utils/imageUtils.ts:

/** * Generates a fallback avatar URL using the ui-avatars.com service. */ export const generateFallbackAvatar = (name: string): string => { return `https://ui-avatars.com/api/?name=${encodeURIComponent( name, )}&background=random`; }; /** * Returns a "safe" avatar URL by validating and sanitizing the input URL. * - Rejects obvious placeholders like `example.com/*` * - Falls back to the ui-avatar generator when URL is missing or invalid */ export const getSafeAvatarUrl = (name: string, url?: string | null): string => { // If no URL provided, generate a fallback if (!url) return generateFallbackAvatar(name); try { // Parse the URL to check its validity and extract the hostname const { hostname } = new URL(url); // Reject common placeholder domains if ( hostname === 'example.com' || hostname.endsWith('.example.com') || hostname === 'localhost' ) { return generateFallbackAvatar(name); } // URL is valid and not a placeholder return url; } catch { // URL is malformed, generate a fallback return generateFallbackAvatar(name); } };

Update image elements in your components like this:

import { getSafeAvatarUrl, generateFallbackAvatar } from '../utils/imageUtils'; // In your component: const avatarUrl = getSafeAvatarUrl(character.name, character.avatar_url); // In your JSX: { (e.target as HTMLImageElement).src = generateFallbackAvatar(character.name); }} />

Sanitize URLs at the repository level:

import { getSafeAvatarUrl } from '../utils/imageUtils'; // In your repository methods: sanitizeCharacter(character: Character): Character { if (!character) return character; // Create a copy to avoid mutating the original const sanitized = { ...character }; // Sanitize avatar URL if present if (sanitized.avatar_url) { sanitized.avatar_url = getSafeAvatarUrl(sanitized.name, sanitized.avatar_url); } // Sanitize feature image URL if present if (sanitized.feature_image_url) { sanitized.feature_image_url = getSafeAvatarUrl(sanitized.name, sanitized.feature_image_url); } return sanitized; }

Database Scan

Check your database for example.com URLs that need to be fixed:

-- SQL query to find all example.com URLs in your database SELECT id, name, avatar_url FROM characters WHERE avatar_url LIKE '%example.com%' OR feature_image_url LIKE '%example.com%';

Then update them with:

-- SQL query to update example.com URLs to fallbacks UPDATE characters SET avatar_url = NULL, feature_image_url = NULL WHERE avatar_url LIKE '%example.com%' OR feature_image_url LIKE '%example.com%';
