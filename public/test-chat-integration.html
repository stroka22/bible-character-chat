If your browser keeps sending you to the old static demo, it’s likely because a cached service-worker is still intercepting requests.

How to Test Chat Functionality

1

Launch the React App

Click one of the buttons above to open the React application.

  • Login Required: Standard flow – you will see the login page first.
  • Skip Login: Appends ?skipAuth=1 so you go straight to character selection.
2

Select a Character

Browse through the available Bible characters and click on any character card. You can use the filters at the top to narrow down your selection:

  • Filter by testament (Old or New)
  • Filter by Bible book
  • Filter by character group
  • Search by name or description
  • Use alphabetical navigation on the right side
3

Start a Conversation

Once you select a character, the interface should transition to the chat view. You'll see:

  • The character's name and avatar at the top
  • An empty chat area (or an opening message from the character)
  • A text input field at the bottom to type your messages
4

Send Messages

Type a message in the input field and press Enter or click the send button. The character should respond based on their biblical persona.

Example questions to try:
  • "Tell me about your life"
  • "What was your greatest challenge?"
  • "How did you encounter God?"
  • "What wisdom would you share with people today?"
5

Explore Character Insights

Click the "Insights" button in the top-right corner of the chat interface to open the insights panel. This panel contains:

  • Historical context about the character
  • Scripture references with hover popups
  • Relationships to other biblical figures
  • Study questions for deeper reflection
6

Return to Character Selection

To chat with a different character, click the "Back" button in the top-left corner of the chat interface. This will return you to the character selection screen.

Troubleshooting

Character Selection Not Working

Symptom: Clicking on a character card doesn't do anything or shows an error.

Solutions:

  • Check the browser console (F12) for any error messages
  • Verify that the ChatContext is properly initialized by looking for "ScalableCharacterSelection MODULE LOADED!" in the console
  • Try refreshing the page and selecting a character again
  • If using a direct URL (like http://localhost:5175?direct=1), try removing the query parameters

Chat Interface Not Appearing

Symptom: After selecting a character, you still see the character selection screen.

Solutions:

  • Check if HomePage.tsx is being used in App.tsx instead of directly rendering ScalableCharacterSelection
  • Verify that the character selection handler is correctly calling the selectCharacter function from ChatContext
  • Try adding ?direct=1 to the URL to bypass routing

Character Not Responding

Symptom: You can send messages, but the character doesn't respond.

Solutions:

  • Check if you have a valid OpenAI API key in your .env file
  • Look for API errors in the browser console
  • Verify that the openai.ts service is properly configured
  • If using a placeholder API key, you'll see a message like "(Character is silent because no OpenAI API key is configured.)"

Blank Screen or Redirects

Symptom: The app shows a blank white screen or keeps redirecting.

Solutions:

  • Use the launch-react-app.sh script to properly start the application
  • Check if service-worker-cleanup.js is properly removing old service workers
  • Clear your browser cache and cookies
  • Try using a different browser or incognito mode
  • If all else fails, use the standalone server with node server.mjs and visit http://localhost:3000

Circular Images Appear Oval

Symptom: Character avatars appear oval-shaped instead of perfectly circular.

Solutions:

  • Verify that the table-based circular image approach is being used in CharacterCard.tsx
  • Ensure the image dimensions are explicitly set to the same width and height (e.g., 150x150px)
  • Check if any CSS is overriding the aspect ratio or border-radius properties
