This will create a view named characters_expected that maps the current database columns to what the React code expects:

  • name → character_name
  • description → short_biography
  • opening_line → opening_sentence

SQL to Execute:

CREATE OR REPLACE VIEW public.characters_expected AS SELECT id, name AS character_name, description AS short_biography, opening_line AS opening_sentence, * -- other cols unchanged FROM public.characters;
