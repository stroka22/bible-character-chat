Environment Configuration

Your API keys should be configured in the .env file in the root of your project:

# Bible Character Chat - Environment Variables # OpenAI API Configuration ***REMOVED*** # Stripe Configuration VITE_STRIPE_PUBLIC_KEY=pk_test_your_stripe_publishable_key_here VITE_STRIPE_PRICE_MONTHLY=price_your_monthly_price_id_here VITE_STRIPE_PRICE_YEARLY=price_your_yearly_price_id_here
Important Security Note: For Stripe, always use a publishable key (starts with pk_test_ or pk_live_), never a secret key.

Common API Issues

  • Invalid API Key Format: Make sure your keys match the expected format
  • Expired or Revoked Keys: Check your account dashboard to ensure keys are active
  • Rate Limiting: You might be hitting API rate limits
  • Network Issues: Ensure your network allows connections to API endpoints
  • CORS Issues: API requests might be blocked by CORS policies
